Проект создания коротких ссылок

Сервис сокращения ссылок с поддержкой ограничения количества переходов, времени жизни (TTL), уникальности ссылок для каждого пользователя, а также с возможностью редактирования и удаления собственных ссылок.
Управление сервисом осуществляется через консольный интерфейс (CLI).
